# 更新日志

## [0.1.0] - 2026-03-07

### 新增
- 项目初始版本《数感乐园》发布
- 核心游戏功能：数字识别、图案对应、选择挑战
- 智能难度调整系统：根据表现自动调整难度
- 儿童友好设计：明亮色彩、友好音效、生动动画
- 错误提示系统：友好的学习提示，无惩罚性反馈
- 中英文双语支持
- 完整的响应式设计：支持手机、平板、桌面
- 游戏设置页面：可调整语言、音效等设置
- 单元测试覆盖核心游戏逻辑

### 技术特性
- React 18 + TypeScript 前端框架
- Vite 构建工具
- Framer Motion 动画库
- react-i18next 国际化
- CSS Modules + CSS Variables 样式方案
- React Context + useReducer 状态管理
- Vitest + React Testing Library 测试框架

### 游戏玩法
1. 显示随机数字（5-20）
2. 生成对应数量的可爱图案（冰淇淋、饼干、星星等）
3. 显示3个连续数字选项，其中一个是正确答案
4. 玩家选择后获得即时视觉和声音反馈
5. 根据表现自动调整难度

### 儿童友好设计要点
- 明亮色彩和圆角元素
- 所有交互元素满足最小44×44像素标准
- 错误时提供友好的学习提示
- 渐进式难度调整，建立儿童信心
- 成功体验建立自信心
